## 2. Gemeinsame Schriftsatz-Anatomie (alle Typen — EIN Rahmen, §10)

Rubrum (Kanzlei → Einschreiben → Gericht/Behörde → Parteien + Vertretung →
«betreffend») → **RECHTSBEGEHREN** (nummeriert; Eventualbegehren-Struktur;
letzte Ziffer immer Kosten/Entschädigung «zzgl. MWST», Auslands-Weiche:
MWST-Zusatz entfällt bei Wohnsitz im Ausland) → ggf. separater Block
**PROZESSUALE ANTRÄGE** (Dispensation, Edition/Stufenklage Art. 85 ZPO,
vorsorgliche Massnahmen) → **BEGRÜNDUNG** zweigeteilt: *I. Formelles*
(Vollmacht → Zuständigkeit → ggf. Klagebewilligung/Fristwahrung) /
*II. Materielles* (ein Abschnitt je Begehren-Komplex: Norm → Subsumtion →
Antrag) → Schlussformel → Mehrfachausfertigung → **separates
Beweismittelverzeichnis**; Beweisofferten als «BO:»-Zeilen direkt unter der
Tatsachenbehauptung (Beilage-Nr. / Edition / Gutachten / Parteibefragung).
→ Dieser Rahmen ist mit unserer Vorlagen-Engine (Bausteine + includeIf +
Felder) vollständig abbildbar; die BO-Mechanik wäre ein NEUER geteilter
Baustein-Typ (Tatsache+Offerte-Paar) — Rahmen-Arbeit nach §10 VOR dem Bau.

## 3. Vorlagen-Typen mit Determinismus-Grenze (Abdeckungskarten-Logik)

### 3.1 Eheschutzgesuch (§§ 72–74) — Vorschlag: ◐ GERÜST mit D-Begehren
- **Begehren-Katalog** (Formel deterministisch / Begründung wertend):
  Getrenntleben (175 ZGB, D) · Obhut allein/alternierend (176 III, 298 ZGB;
  Struktur D, Inhalt W) · persönlicher Verkehr mit «gerichtsüblichem»
  Raster (273 ZGB; Raster D: 2.-WE/Feiertags-Alternanz/Ferienwochen/
  Ankündigungsfrist) · Bar- + BETREUUNGS-Unterhalt als GETRENNTE Begehren
  (276/285 ZGB; Formel D «monatlich im Voraus auf den Ersten», Betrag W —
  NIE rechnen, §2) · Ehegattenunterhalt (163/176 ZGB; Rückwirkung max. 1 J.
  173 III) · Wohnungszuteilung mit Auszugsfrist (176 I Ziff. 2, D-Raster) ·
  Schuldneranweisung an Arbeitgeber (177 ZGB, weitgehend D) · Rayonverbot
  (172 III i.V.m. 28b ZGB; Begehren D mit Plan-Beilage, Gefährdung W) ·
  Verfügungsbeschränkung/Kontosperre (178 ZGB; Vermögenswert INDIVIDUELL
  bezeichnen — keine Gesamtsperre) · Gütertrennung (176 I Ziff. 3, D) ·
  Kostenfolge + optional PKV vor URP (159 III/163 ZGB vor 117 ZPO).
- **Prozess-Anker:** summarisch 271 lit. a ZPO (kein Stillstand! unsere
  314-Abs.-2-Weiche = Rechtsmittel), keine Schlichtung (198 lit. a),
  zwingend Wohnsitz eines Ehegatten (Art. 23 ZPO), Glaubhaftmachung,
  Kinderbelange 296 ZPO.
- **Beilagen-Taxonomie** (Checklisten-tauglich, D): Vollmacht · Lohn/Bonus ·
  Steuern · Wohnkosten · KK-Policen OKP/VVG · Gesundheits-/Schul-/ÖV-Belege ·
  bei Gewaltschutz: Polizeirapporte/Arztzeugnisse/Plan.
- **Varianten-Weichen:** knapp (Manko-Ausweis 301a lit. c ZPO/286a ZGB,
  betreibungsrechtliches Existenzminimum) vs. überdurchschnittlich
  (familienrechtliches Existenzminimum, Sparquote, 4-Spalten-Bedarfstabelle)
  vs. Durchsetzungs-Zweitgesuch (nur Anweisung/Rayon/Sperre; Superprovisorik
  265 ZPO).

### 3.2 Gemeinsames Scheidungsbegehren (§§ 75/76) — Vorschlag: ◐ GERÜST
- Zwei Unter-Typen: UNBEGRÜNDET (Anträge nur orientierend, 286 II ZPO) und
  BEGRÜNDET (mit Teilvereinbarung; Genehmigungsbegehren 279 ZPO).
- Pflicht-Mechanik (D): Erklärung auf gerichtliche Beurteilung der
  streitigen Folgen muss EXPLIZIT sein (112 I ZGB/286 I ZPO, sonst
  Nachfrist); Anhörungs-Kaskade (287 ZPO: Bestätigung → Bindung; Widerruf
  → Abschreibung; Teilstreit → kontradiktisch 288 II).
- Antrags-Gliederung als festes Raster (D): Sorge → Obhut → Verkehr →
  Kinderunterhalt (Bar/Betreuung getrennt, Altersstaffel, Volljährigen-
  klausel, a.o.-Kosten-Quote) → Ehegattenunterhalt (inkl. Negativ-Antrag)
  → Indexierung (Sammelziffer) → Erziehungsgutschriften (29sexies AHVG,
  Versicherten-Weiche) → Vorsorgeausgleich (122 f. ZGB, hälftig + FZG-22-
  Anweisung; WEF-Komplex 30c VI BVG mit Grundbuch-Zusatzbegehren) →
  Güterrecht (unbeziffert mit Bezifferung nach Edition, Art. 85 ZPO) →
  Edition → Kosten.
- **Einleitungsform-WEICHE** (D, decision-tree): beide willig + Volleinigung
  → 111; beide willig + Teileinigung → 112; einer willig + ≥2 J. getrennt
  → Klage 114; <2 J. → 115 (streng) oder 114 mit Einlassungs-Risiko.

### 3.4 Vorsorgliche Massnahmen / Abänderung / Ergänzung (§§ 81–82a)
